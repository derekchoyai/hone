# Worked Examples

Each example shows a real-shaped piece of AI-assisted work, the human's interview answers,
and the resulting score with analysis. This is the fastest way to understand what Hone
actually does. (These examples predate Delegation assessment — their interviews probe only
Discernment, so each score shown is the AI-Q with D1 alone assessed, which is identical
math. See [`spec/aiq.md`](../spec/aiq.md).)

| Example | Domain | Illustrates |
|---|---|---|
| [coding-review.md](coding-review.md) | coding | High-quality code, **low** judgment — the dangerous case |
| [strategy-memo.md](strategy-memo.md) | strategy | An unchecked load-bearing assumption |
| [research-summary.md](research-summary.md) | research | Confident-but-unsupported claims |

The recurring lesson: **a great output handled with low judgment scores low** — which is
exactly the case model-quality evals can't see.
